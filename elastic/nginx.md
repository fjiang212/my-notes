## Install trough yum repository
https://www.nginx.com/resources/wiki/start/topics/tutorials/install/
```
[nginx]
name=nginx repo
baseurl=http://nginx.org/packages/centos/7/$basearch/
gpgcheck=0
enabled=1
```
yum install nginx
